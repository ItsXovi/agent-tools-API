PDF processing API

PDF Toolkit API

Merge, split, compress, and watermark PDF files. Upload via multipart form data, get back processed PDFs.

Endpoints

  • POST /v1/pdf/merge — merge multiple PDFs
  • POST /v1/pdf/split — split by page ranges (pages=1-3,5 or all)
  • POST /v1/pdf/compress — compress a PDF
  • POST /v1/pdf/watermark — add text watermark
  • GET /v1/usage — monthly usage for your API key

Merge

curl -X POST https://YOUR_DOMAIN/v1/pdf/merge \
  -H "X-API-Key: atk_your_key_here" \
  -F "files=@a.pdf" \
  -F "files=@b.pdf" \
  --output merged.pdf

Split

curl -X POST "https://YOUR_DOMAIN/v1/pdf/split?pages=1-3" \
  -H "X-API-Key: atk_your_key_here" \
  -F "file=@document.pdf" \
  --output pages.pdf

Compress

curl -X POST https://YOUR_DOMAIN/v1/pdf/compress \
  -H "X-API-Key: atk_your_key_here" \
  -F "file=@large.pdf" \
  --output compressed.pdf

Watermark

curl -X POST "https://YOUR_DOMAIN/v1/pdf/watermark?text=DRAFT&position=diagonal" \
  -H "X-API-Key: atk_your_key_here" \
  -F "file=@document.pdf" \
  --output watermarked.pdf
